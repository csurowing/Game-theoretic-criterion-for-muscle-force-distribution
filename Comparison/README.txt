This folder contains the files necessary to perform the comparison between the extended quadratic/game-theoretic criteria with the results of the paper by Grimmelsmann et.al.
The code here is self-contained. Just make sure ComparisonData.m is in the folder or search path and run RunComparison.m.

Figures as in Richter and Mastropieri, J. Biomech. Open, 2026 will be created.
